COMO ALINHAR TEXTO EM HTML?

HTML significa HyperText Markup Language. Ele é usado para projetar páginas da web usando uma linguagem de marcação. HTML é a combinação de hipertexto e linguagem de marcação. O hipertexto define o link entre as páginas da web. Uma linguagem de marcação é usada para definir o documento de texto dentro da tag que define a estrutura das páginas da web.

HTML é usado pelo navegador para manipular texto, imagens e outros conteúdos para exibi-los no formato necessário.
